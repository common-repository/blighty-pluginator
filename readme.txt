This plugin monitors installed WordPress plugins on your installation for updates and notifies via email when they are available.

== Description ==
This plugin monitors installed WordPress plugins on your installation. It notifies the admin (or other email) when updates are available. Plugins can be excluded from monitoring.

== Installation ==
- Install and optionally set a notification email address via the settings.
- Optionally enable the "snapshot" feature. This monitors plugins and reports by email when they are activated or deactivated.
- That's it.
